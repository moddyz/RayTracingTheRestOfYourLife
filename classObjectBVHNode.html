Detailed Description

ObjectBVHNode is a single node in the bounding volume hierarchy.

Constructing a root ObjectBVHNode with an array of SceneObject(s) will result in the recursive construction of the BVH tree, with the leaf nodes representing the original SceneObject(s).

BVH is represented as a binary tree, so each ObjectBVHNode has left & right children.

This BVH utilizes a object-based partioning strategy, by sorting the object range based on their extent minima with respect to a randomly chosen axis then splitting down the middle.

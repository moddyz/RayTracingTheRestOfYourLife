Detailed Description

SpatialBVHNode is a single node in the bounding volume hierarchy.

Constructing a root SpatialBVHNode with an array of SceneObject(s) will result in the recursive construction of the BVH tree, with the leaf nodes representing the original SceneObject(s).

BVH is represented as a binary tree, so each SpatialBVHNode has left & right children.

This BVH utilizes a spatial partioning strategy, by evenly splitting the input volume across its longest axis, then partitioning the objects into either of the halves.

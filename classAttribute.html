Detailed Description

template<typename ValueT>
class Attribute< ValueT >

Attribute encapsulates a collection of time sampled values of ValueT, allowing efficient value retrieval or interpolation for a requested time.

Template Parameters
ValueTThe underlying value type that this attribute
Precondition
ValueT must be default constructable, with deterministic value(s).
ValueT must be also be linearly interpolatable via gm::LinearInterpolation.

◆ Value()

template<typename ValueT>
ValueT Attribute< ValueT >::Value ( float  i_time = 0.0f) const
inline

Get the value at time i_time.

The value is sampled through the following ordered cases:

  1. If this attribute has a single time sample, then that value is returned.
  2. If this attribute has no time samples, then the value type is default constructed and returned.
  3. Otherwise, a binary search will be performed to find the requested time sampled value.
    • If the requested i_time is not stored by this attribute, the value will be linearly interpolated by the nearest lower and upper time sampled values.
    • If i_time is less than the lowest time then the value at the lowest time will be returned.
    • The same scenario applies for the scenario when i_time is greater than the highest/ time, the highest time sampled value is returned!
Parameters
i_timeThe time at which to sample to value.
Returns
The time sampled value.
